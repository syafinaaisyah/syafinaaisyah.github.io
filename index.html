🧪 Labs

Lab 1 · UI

User Interface & Layouts
Reflection: This lab introduced me to Jetpack Compose fundamentals. I learned to arrange UI elements with Row, Column, and Box, and gained confidence in building responsive layouts that adapt to different screen sizes.

Lab 2 · Interaction

Event handling & state
Reflection: Working with click events and text input taught me how to manage state in Compose. I practiced using remember and mutableStateOf to create interactive forms and dynamic UI updates.

Lab 3 · Material Design

Material 3 theming
Reflection: I explored Material 3 components such as Card, Button, and TopAppBar. Applying custom colour schemes and typography helped me understand how to create a cohesive, accessible visual identity for health-related apps.

Lab 4 · Navigation

Jetpack Navigation
Reflection: Implementing NavHost and screen transitions was a key milestone. I learned to pass arguments between destinations and handle back stack behaviour, which is essential for multi-screen health tracking apps.

Lab 5 · Room

Local persistence
Reflection: Setting up Room with entities, DAOs, and coroutines gave me practical experience with offline data storage. I now feel comfortable performing CRUD operations and querying relational data asynchronously.

📱 Projects

FitTrack · Project 1

Lab 4 + Working Features

Problem: Many individuals struggle to maintain consistent fitness routines because they lack a simple, intuitive way to log daily activities and monitor their progress over time.

Lessons learned: Building FitTrack taught me to combine navigation with real-time data entry and validation. I improved my skills in state management and learned to design a user-friendly dashboard that visualises health metrics clearly for better decision-making.

FitTrack · Project 2

Lab 5 + API + Sensors + Cloud

Problem: Users need a holistic health view that combines manual activity logs with automatic sensor data and secure cloud backup to ensure their progress is never lost and can be accessed anywhere.

Lessons learned: Integrating Retrofit, Room, and Firebase Firestore showed me the power of a layered architecture. I learned to handle sensor data (step counter) and sync it with the cloud while maintaining a responsive UI and handling offline scenarios gracefully.
